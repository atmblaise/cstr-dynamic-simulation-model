# cstr-dynamic-simulation-model
# Continuous Stirred Tank Reactor (CSTR) Dynamic Modeling and Simulation

## Overview

This project presents a dynamic simulation of a Continuous Stirred Tank Reactor (CSTR) developed using Python. The objective is to model reactor behavior over time by applying fundamental chemical engineering principles including mass balances, reaction kinetics, and numerical integration techniques.

The model provides insight into how reactor concentration and temperature evolve under changing operating conditions and can serve as a foundation for process control, optimization, and digital twin development.

---

## Objectives

* Develop a first-principles dynamic model of a Continuous Stirred Tank Reactor.
* Simulate transient concentration and temperature behavior.
* Investigate the effect of operating conditions on reactor performance.
* Demonstrate the application of numerical methods to chemical process systems.
* Provide a reusable framework for future process control and optimization studies.

## Model Description

The reactor is assumed to be perfectly mixed, resulting in uniform concentration and temperature throughout the vessel.

The governing equations are derived from:

* Species mass balances
* Reaction rate expressions
* Energy balances (when thermal effects are included)

The resulting ordinary differential equations (ODEs) are solved numerically to obtain the dynamic response of the system.
